## Getting Started

### Your Role

As Office Staff, you support sales and field teams with:
- ✅ Call logging and follow-up tracking
- ✅ Event and task scheduling
- ✅ Document organization and filing
- ✅ Report generation for management
- ✅ Customer inquiries and support
- ✅ Data entry and cleanup

---

## Call Logging

### Log an Inbound Call

1. Navigate to **Call Logs**
2. Click **+ Log Call**
3. Fill in call details:
   - **Direction**: Inbound
   - **Phone Number**: Caller's number
   - **Contact**: Search and select existing contact
     - If new, click **+ Create Contact**
   - **Duration**: Call length (min:sec)
   - **Outcome**: Answered, Voicemail, No Answer
   - **Disposition**: Interested, Not Interested, Call Back
   - **Notes**: Call summary and action items
4. Click **Save Call Log**

**Tip**: Log calls immediately while details are fresh.

---

### Log an Outbound Call

Same as inbound, but select **Direction**: **Outbound**

---

### Link Call to Project

If call relates to existing project:

1. When logging call, click **Project** dropdown
2. Search for project by name or contact
3. Select project
4. Call appears in project's activity feed

---

### Listen to Call Recordings

If Twilio call recording is enabled:

1. Go to **Call Logs**
2. Find call with **recording available** icon (microphone)
3. Click call to open details
4. **Call Recording** section shows audio player
5. Use player controls:
   - Play/Pause
   - Seek through recording
   - Adjust speed (1x, 1.5x, 2x)
   - Download recording

**Privacy Note**: Inform callers of recording per state laws.

## Event & Task Management

### Create an Event

Schedule meetings, site visits, appointments.

1. Navigate to **Calendar**
2. Click **+ New Event**
3. Fill in event details:
   - **Title**: e.g., "Site visit - Smith property"
   - **Type**: Meeting, Site Visit, Call, Other
   - **Date & Time**: Start and end
   - **Location**: Address or "Office" or "Phone"
   - **Attendees**: Select team members
   - **Contact/Project**: Link to contact or project
   - **Notes**: Agenda or special instructions
4. Click **Create Event**

Event appears on team calendars.

---

### View Team Calendar

1. Go to **Calendar** view
2. Switch between views:
   - **Day**: Today's schedule
   - **Week**: 7-day view
   - **Month**: Monthly overview
3. Click any event to see details
4. Edit or delete as needed

---

### Create a Task

Assign follow-ups and to-dos.

1. Navigate to **Tasks**
2. Click **+ New Task**
3. Fill in task details:
   - **Title**: Task description
   - **Assigned To**: Team member
   - **Due Date**: Deadline
   - **Priority**: High, Medium, Low
   - **Contact/Project**: Link to relevant record
   - **Notes**: Additional context
4. Click **Create Task**

Assignee receives notification.

---

### Track Task Completion

1. Go to **Tasks** view
2. Filter by:
   - **Assigned To**: Specific person
   - **Status**: Pending, In Progress, Complete
   - **Due Date**: Today, This Week, Overdue
3. Mark tasks complete with checkbox
4. View completion stats

## Document Organization

### Upload Documents

1. Navigate to contact or project
2. Go to **Documents** tab
3. Click **+ Upload**
4. Select file(s) from computer
5. For each file, enter:
   - **Document Name**: Descriptive title
   - **Type**: Contract, Estimate, Photo, Invoice, etc.
   - **Description**: Optional notes
6. Click **Upload**

Files associate with that contact/project.

---

### Organize with Folders

1. In Documents tab, click **+ New Folder**
2. Name folder:
   - "2025 Contracts"
   - "Insurance Claims"
   - "Before/After Photos"
3. Drag documents into folders
4. Create subfolders as needed

---

### Search Documents

1. Go to **Documents** (global search)
2. Use search filters:
   - **File Name**: Enter keywords
   - **Type**: Filter by document type
   - **Date Range**: Uploaded between dates
   - **Contact**: Specific customer
3. Results show matching documents
4. Click to view or download

---

### Share Documents

#### Via Email

1. Find document
2. Click **Share** → **Email**
3. Enter recipient email
4. Add message (optional)
5. Click **Send**

Document attached to email.

#### Via Link

1. Find document
2. Click **Share** → **Generate Link**
3. Copy shareable link
4. Send link via email, SMS, etc.

**Security**: Links expire after 7 days.

## Report Generation

### Daily Activity Report

Shows team activity for the day.

1. Navigate to **Reports** → **Daily Activity**
2. Select date
3. Report shows:
   - Calls logged
   - Knocks logged
   - Leads created
   - Emails sent
   - Tasks completed
4. Export to PDF or CSV

---

### Pipeline Report

View sales pipeline health.

1. Go to **Reports** → **Pipeline**
2. Select date range
3. Report shows:
   - Projects by stage
   - Total pipeline value
   - Average deal size
   - Days in each stage
   - Conversion rates
4. Filter by sales rep
5. Export report

---

### Contact Activity Report

Track contact engagement.

1. Go to **Reports** → **Contact Activity**
2. Select date range
3. Report shows:
   - New contacts added
   - Emails sent/received
   - Calls logged
   - Meetings scheduled
   - Last contact date
4. Filter by contact type
5. Export to CSV for analysis

---

### Custom Reports

1. Navigate to **Reports** → **Custom**
2. Click **+ New Report**
3. Select data source:
   - Contacts
   - Projects
   - Activities
   - Call Logs
4. Choose fields to include
5. Add filters and sorting
6. Save report template
7. Run anytime or schedule automatic delivery

## Customer Support

### Handle Inbound Inquiries

#### Email Inquiries

1. Check **support@yourcompany.com** inbox
2. For each inquiry:
   - Search for existing contact
   - If new, create contact
   - Log activity in CRM
   - Respond to customer
   - Create task for follow-up if needed

#### Phone Inquiries

1. Answer call professionally
2. Log call in CRM (see Call Logging)
3. Search for contact by phone/name
4. Address question or route to appropriate person
5. Create task for any follow-up needed

---

### Create Support Tickets

For complex issues requiring team collaboration:

1. Go to **Tasks**
2. Click **+ New Task**
3. Set **Type**: Support Ticket
4. Fill in details:
   - **Title**: Brief issue description
   - **Priority**: Based on urgency
   - **Assigned To**: Appropriate team member
   - **Contact**: Link to customer
   - **Notes**: Detailed issue description
5. Click **Create**

Track ticket status in Tasks view.

## Best Practices

### Call Logging

✅ **Do**:
- Log every call immediately
- Use consistent disposition labels
- Add detailed notes with action items
- Link calls to contacts and projects
- Set follow-up reminders

❌ **Don't**:
- Batch log calls hours later (details forgotten)
- Leave notes field blank
- Forget to link to contact record
- Skip logging quick calls

---

### Event Scheduling

✅ **Do**:
- Send calendar invites to attendees
- Include location and dial-in details
- Set reminders (15 min, 1 day before)
- Add prep notes or agenda
- Update if rescheduled

❌ **Don't**:
- Schedule without checking availability
- Forget to invite required attendees
- Leave location field blank
- Cancel without notification

---

### Document Management

✅ **Do**:
- Use descriptive file names
- Organize into logical folders
- Delete outdated/duplicate files
- Tag documents for easy search
- Set folder permissions appropriately

❌ **Don't**:
- Upload files with generic names ("Document1.pdf")
- Keep everything in root folder
- Upload duplicate copies
- Share confidential docs publicly

---

### Report Generation

✅ **Do**:
- Run reports consistently (weekly, monthly)
- Share reports with relevant stakeholders
- Review trends and patterns
- Export data for deeper analysis
- Archive historical reports

❌ **Don't**:
- Generate reports inconsistently
- Ignore anomalies or outliers
- Share raw data without context
- Delete historical data

## Troubleshooting

### Can't Find Contact

**Solutions**:
- Try searching by phone, email, or partial name
- Check for spelling errors
- Look in "Archived" contacts
- Create new contact if truly doesn't exist

### Report Not Loading

**Solutions**:
- Check date range isn't too large
- Reduce filters to simplify query
- Refresh page
- Export to CSV instead of viewing online

### Document Upload Failing

**Solutions**:
- Check file size (max 10MB)
- Verify file type is allowed
- Try different browser
- Check internet connection
