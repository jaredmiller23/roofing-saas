Quick Reference - Common Tasks

Add a New Lead

  1. Go to Contacts
  2. Click + Add Contact
  3. Fill in their information
  4. Click Create Contact

Log a Door Knock

  1. Go to Knock
  2. Click Log Knock
  3. Click Get My Location
  4. Select the result (Interested, Not Home, etc.)
  5. Click Log Knock

Send a Contract for Signature

  1. Go to Signatures
  2. Click + New Document
  3. Choose a template or start fresh
  4. Fill in the details
  5. Add signature fields
  6. Click Create Document
  7. Click Send and enter their email

Start a Claim Inspection

  1. Go to the Project
  2. Click Claims
  3. Click Start Inspection
  4. Follow the wizard steps
  5. Submit when done

Move a Project to the Next Stage

  1. Go to Pipeline
  2. Find the project card
  3. Drag it to the next column
  4. Or click the card and use the stage buttons
